# band-names-app
Real time app create in react and nodejs with socket.io

## Backend

### Installation
```bash
yarn install
```

### Run project
```bash
yarn run dev
```

## Frontend

### Configuration
-Frontend
In band-names-front create .env file, after copy and paste from .env.example in .env

### Installation
```bash
yarn install
```

### Run project
```bash
yarn start
```
